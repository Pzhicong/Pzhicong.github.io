防抖与节流

发表于 2017-04-26 | 分类于 前端
窗口的resize、scroll、输入框内容校验等操作时，采用debounce（防抖）和throttle（节流）的方式来减少触发的频率。

防抖动（debouce）

若干个函数调用合成 一次，并在给定时间过去之后仅被调用一次

节流（throttle）

节流函数允许一个函数在规定的时间内只执行一次
